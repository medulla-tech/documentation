This guide has been designed to facilitate the learning process of Medulla. It presents the main features of Medulla and the basic actions to perform to use Medulla effectively.

Guide Scope
==============================

This guide is intended for a quick introduction to Medulla. It will enable you to use the major features of Medulla for managing the workstations in your environment.

Functional Overview
============================

The Medulla product is a suite of IT asset management tools grouped within a single interface. Medulla includes a set of value-added services (VAS) such as inventory, imaging, broadcasting, remote desktop access, data backup/restore, and monitoring.

Medulla provides a user interface entirely dedicated to setting up automation policies for IT tasks, allowing the complete management of workstation lifecycle and improving security. Medulla is directly accessible via a standard web interface to remain accessible to mainstream browsers.
